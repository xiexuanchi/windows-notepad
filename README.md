# NotePad (Python Tkinter)

一个使用 Python 和 Tkinter 编写的轻量级、响应式文本编辑器。它具有简洁的界面、实用的状态栏以及跨平台的快捷键支持。

## ✨ 功能特性

* **基础编辑**：支持新建、打开、编辑和保存文本文件（默认为 `.txt`）。
* **状态栏实时更新**：右下角实时显示光标所在的行号（Line）和列号（Col）。
* **现代视觉体验**：
* 使用 **Consolas** 字体，字号设置为 16，适合代码或文本阅读。
* 增加了文本内边距（Padding），视觉感更舒适。


* **撤销重做**：内置撤销（Undo）功能，方便纠正输入错误。
* **跨平台快捷键**：自动检测操作系统，在 Windows/Linux 上使用 `Ctrl` 组合键，在 macOS 上使用 `Command` 组合键。

---

## ⌨️ 快捷键指南

| 动作 | Windows / Linux | macOS (Darwin) |
| --- | --- | --- |
| **新建文件** | `Ctrl + N` | `Cmd + N` |
| **打开文件** | `Ctrl + O` | `Cmd + O` |
| **保存文件** | `Ctrl + S` | `Cmd + S` |
| **退出应用** | `Ctrl + Q` | `Cmd + Q` |
| **撤销操作** | `Ctrl + Z` | `Cmd + Z` |

## 🚀 快速开始

### 前置条件

确保你的系统中已安装 Python 3。Tkinter 通常随 Python 标准库一起安装。

### 运行步骤

1. 下载或克隆本仓库。
2. 在终端或命令行中进入文件所在目录。
3. 运行以下命令：
```bash
python main.py

```

## 📝 代码实现细节说明

* **防冗余保存**：在保存文件时使用了 `end-1c` 参数，有效防止 Tkinter 默认在文末添加额外的换行符。
* **异常处理**：内置了文件读写的 `try-except` 块，防止因权限或文件损坏导致的程序崩溃。
* **自动重置**：在新建或打开文件后，会自动清空撤销历史记录（`edit_reset`），确保操作逻辑清晰。
